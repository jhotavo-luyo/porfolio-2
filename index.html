Backend

API RESTful construida sobre Node.js y Express, diseñada para ser el servidor central del sistema. Sigue una arquitectura en capas (Router, Controller, Service, Repository) que separa claramente las responsabilidades: manejo de rutas, lógica de negocio y acceso a la base de datos (MySQL). Esta estructura modular facilita el mantenimiento y la escalabilidad. La seguridad es un componente clave, implementando middlewares para la autenticación de usuarios vía JWT y la gestión de permisos por roles.


Frontend

Aplicación web desarrollada con React y Vite que funciona como una SPA (Single-Page Application). Para el diseño de la interfaz se utiliza Tailwind CSS, y la navegación entre páginas se gestiona con React Router. La arquitectura separa la vista pública (con un modelo 3D interactivo) de un panel de administración privado, cuyo acceso está protegido por JSON Web Tokens (JWT) para garantizar la seguridad.
